一、订阅链接配置进阶

1.1 多订阅源配置

为了确保节点的稳定性和可用性，建议配置多个订阅源：

配置步骤：

  1. 打开 V2rayN → 订阅分组设置
  2. 点击「添加」创建多个订阅分组
  3. 分别填入不同订阅源的地址
  4. 设置不同的更新间隔（如 12小时、24小时）
  5. 全部设置为「自动更新」

五、一键测速功能

5.1 V2rayN 内置测速

使用方法：

  1. 在服务器列表中选择要测速的节点
  2. 右键点击选择「测量延迟」或「全选测量」
  3. 等待测速完成（通常 5-30 秒）
  4. 查看延迟结果

测速结果说明：

延迟范围 评价 适用场景
< 50ms ⭐⭐⭐⭐⭐ 极佳 游戏、视频通话
50-100ms ⭐⭐⭐⭐ 良好 在线视频、浏览
100-200ms ⭐⭐⭐ 一般 日常浏览
200-500ms ⭐⭐ 较差 仅文字浏览
> 500ms ⭐ 极差 不建议使用

5.2 使用 Speedtest 测速

在线测速网站：

网站 地址
Speedtest https://www.speedtest.net/
Fast.com https://fast.com/
Cloudflare Speedtest https://speed.cloudflare.com/

测速步骤：

  1. 确保 V2rayN 已连接节点
  2. 开启系统代理（PAC 或全局模式）
  3. 打开浏览器访问测速网站
  4. 点击开始测速
  5. 记录下载/上传速度

  3. 重启客户端

  5. 更新完成后关闭加速

十、常见问题与解决方案

10.1 连接不稳定

排查步骤：

  1. 检查本地网络是否稳定
  2. 更换延迟更低的节点
  3. 尝试不同的传输协议（TCP/WS/mKCP）
  4. 检查是否有防火墙拦截
  5. 重启 V2rayN 客户端

10.2 速度慢

优化建议：

  1. 选择地理位置更近的节点
  2. 切换到更快的传输协议
  3. 开启 MTC TCP 优化
  4. 关闭不必要的后台进程
  5. 检查是否开启了带宽限制

10.3 DNS 泄露

检测方法：

  1. 访问 https://dnsleak.com/
  2. 查看显示的 DNS 服务器
  3. 如果显示本地 ISP DNS，说明存在泄露

解决方案：

  1. 开启 DNS 加密
  2. 使用远程 DNS
  3. 配置正确的路由规则

10.4 IPv6 问题

问题：部分节点不支持 IPv6

解决方案：

  1. 在系统设置中禁用 IPv6
  2. 或在路由规则中屏蔽 IPv6 流量

十一、安全建议

11.1 隐私保护

  • 🔐 定期更换节点密码
  • 🔐 使用强加密协议（VLESS/XTLS）
  • 🔐 开启 TLS 传输
  • 🔐 避免使用不信任的免费节点

11.2 防止泄露

  • 🛡️ 启用域名分流
  • 🛡️ 防止 DNS 泄露
  • 🛡️ 阻止 IPv6 流量
  • 🛡️ 定期检查连接日志

11.3 合规使用

  • ⚖️ 遵守当地法律法规
